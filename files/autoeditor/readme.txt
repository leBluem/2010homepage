AutoEditor 1.1 - 30. Mar 2012 - http://blumetools.dd-dns.de/

Freeware program for automatic text manipulation via masks and counters. Features: search and replace, open save, browse ... usefull for fast generating of a lot of text. This program was coded fast und dirty and comes with no doc. 

Uses SynEdit Component from http://synedit.sourceforge.net
